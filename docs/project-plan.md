# 기획서 — 어디갈지도 (Eodigaljido)

---

## 1. 서비스 개요

**어디갈지도** – 친구와 함께 만드는 실시간 공동 여행 루트 플래너

| 항목 | 내용 |
|---|---|
| 서비스 한 줄 소개 | 친구들과 실시간으로 여행 경로를 함께 만들고, 채팅하며, 위치 공유까지 하는 모바일 애플리케이션 |
| 핵심 가치 제안 | "메신저와 지도를 따로 짜는 여행은 지저분하다" → 한 곳에서 깔끔하게 해결 |
| 타겟 사용자 | 20~39세 MZ세대 / 친구·연인·소모임 단위 여행자 (3~8명 내외 그룹) |
| 주요 차별점 | 1. 지도 기반 실시간 멀티 편집 <br> 2. 채팅 ↔ 지도 양방향 연동 <br> 3. 타 사용자의 추천루트 공유 가능 |

---

## 2. 서비스 목적 및 비전

- **단기 목적** : 카카오톡, 디스코드, 네이버 지도, 구글맵 등 메신저와 지도 서비스가 뒤섞인 불편한 여행 계획 경험을 통합하여 사용자 편의성 증대

---

## 3. 사용자 여정 — 핵심 플로우

1. 회원가입 (휴대폰 인증 or SNS)
2. 성향 설문조사 온보딩 (5~6단계) → 초기 맞춤 추천 루트 제시
3. 홈 화면 → 새 루트 생성 or 기존 루트 참여
4. 루트 편집 화면 (지도 + 리스트 + 채팅)
5. 실제 여행 중 → 실시간 위치 공유 + 체크인
6. 여행 후 → 이동 기록

## 5. 화면 구성 및 주요 플로우

### 5-1. 온보딩 & 가입

- Splash → 로고 + "어디갈지도"
- 이용약관 → 닉네임·프로필 사진 설정
- 성향 설문 5~6단계 (지역 / 나이 / 활동 / 성별)
- 설문 완료 → "님을 위한 추천 코스 3개" 바로 제시

### 5-2. 홈 & 루트 목록

- 상단 탭 : 홈 / 공동 루트 / 내 코스 / 채팅 / 전체
- 추천 섹션 (초기 1~2주)
- 내가 만든/참여 중인 루트 카드 리스트
  - 썸네일 지도 스냅샷
  - 제목, 기간, 참여자 아바타들
  - 마지막 수정 시간·사람

### 5-3. 루트 편집 화면 (가장 핵심)

- 상단 : 루트 제목 / 공유 버튼 / 참여자 목록 (아바타)
- 좌측 또는 하단 시트 : 경유지 리스트 (드래그 정렬 가능)
- 중앙 : 지도 (네이버/카카오/구글 Maps SDK 중 선택)
  - 출발·경유·도착 핀 (색상 = 사용자 색상)
  - 실시간 편집 중인 사람 아바타 떠다님
- 우측 또는 하단 시트 : 채팅창 (루트 전용 채팅방 자동 생성)

### 5-4. 채팅 ↔ 지도 연동 `[보류]`

- 채팅에 "강남역 1번 출구" 입력 → 자동 인식 → 지도에 임시 핀 생성
- 핀 클릭 → "이 장소 경유지로 추가" 버튼
- 추가 후 채팅에 자동 메시지 "김XX님이 강남역 1번 출구를 경유지에 추가했어요"
