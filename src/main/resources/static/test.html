카카오 로그인 / 회원가입

POST/auth/oauth/kakao
버튼 클릭 → 카카오 로그인 페이지로 이동 → 완료 시 이 페이지로 돌아와 자동으로 API를 호출합니다.
기존에 이메일 계정과 카카오를 연동한 경우에도 이 버튼으로 로그인됩니다.

카카오 계정 연동

POST/auth/oauth/kakao/link   DELETE/auth/oauth/kakao/link
이메일(LOCAL)로 가입한 계정에 카카오를 연동합니다.
연동 후에는 카카오 로그인으로도 이 계정에 접근할 수 있습니다.
먼저 로그인(Access Token)이 필요합니다.
🔗 연동 상태: 미확인 (프로필 조회 필요)
